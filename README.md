## 进程的组成
一个进程应该包括：
-	程序的代码；
-	程序处理的数据；
-	程序计数器中的值，指示下一条将运行的指令；
-	一组通用的寄存器的当前值，堆、栈；
-	一组系统资源（如打开的文件）

总之，进程包含了正在运行的一个程序的所有状态信息。

## 进程的特点
	动态性：可动态地创建、结束进程；
	并发性：进程可以被独立地调度并占用处理机运行
	独立性：不同进程的工作不相互影响；
	制约性：因访问共享数据/资源或进程间同步而产生制约。
